What is Pool Type?

Pool Type is a classification code that describes the physical conditions of a swimming pool.
It is defined using five criteria: location, depth, water type, number of lanes, and lane length.


An outdoor pool with a depth of 2 meters, freshwater, 6 lanes, and a lane length of 50 meters is classified as

#ODF6L50

Location

O — Outdoor
I — Indoor

Depth

S — Shallow (less than 1.7 m)
D — Deep (1.7 m or deeper)

Water Type

S — Saltwater
F — Freshwater

Lane Count

Number + L (e.g., 6L = 6 lanes)

Lane Length

Number in meters (e.g., 50 = 50 meters)
